# Fnkle Snake

## Summary
A simple game for practicing my C++ skills as well as messing around with SDL2 while having fun.

## Current Features
- Snake
  - Snake Movement
  - Snake Collision
  - Snake can Eat Fruit
- Fruit
  - Fruit Spawning
  - Limits number of fruit to value in code
- Game borders
  - Game Over on collision
